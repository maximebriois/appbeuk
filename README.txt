APPLICATION NAHEULBEUK — MISE EN LIGNE

MÉTHODE LA PLUS SIMPLE : CLOUDFLARE PAGES
1. Crée un compte gratuit sur Cloudflare.
2. Ouvre « Workers & Pages ».
3. Clique « Create » puis « Pages » puis « Upload assets » / « Direct Upload ».
4. Donne un nom au projet, par exemple : naheulbeuk-audio.
5. Décompresse le ZIP fourni.
6. Dépose TOUT LE CONTENU du dossier « naheulbeuk-audio » (index.html doit être à la racine).
7. Clique sur « Deploy site ».
8. Cloudflare fournit une adresse du type https://naheulbeuk-audio.pages.dev

IMPORTANT
- Ne dépose pas seulement le fichier ZIP si l'écran demande les fichiers du site : décompresse-le d'abord.
- index.html, app.js, style.css, episodes.js, manifest.json, sw.js, logo.png et le dossier icons doivent rester ensemble.
- Les MP3 et images restent sur Cloudinary.

MISE À JOUR PLUS TARD
- Modifie episodes.js pour ajouter/corriger un titre, une image ou un MP3.
- Dans Cloudflare Pages, crée un nouveau déploiement et redépose tous les fichiers.

INSTALLATION SUR TÉLÉPHONE
- Android/Chrome : menu ⋮ > Ajouter à l'écran d'accueil / Installer l'application.
- iPhone/Safari : Partager > Sur l'écran d'accueil.
